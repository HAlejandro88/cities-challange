## Intalaciones

- instalar **node js**
- instalar **MongoDB**

Descargar los paquetes necesarios
> npm install

Guardar los datos en la base de datos
> npm run seed

Para Ejecutar el proyecto en produccion
> npm start

Para Ejecutar el proyecto en desarrollo
> npm run dev

Ejecutar pruebas 
> npm run test
